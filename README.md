# Introduction to Databases

Database Systems project developed during the Computer Science Bachelor at the University of Basel.

## Overview

This repository contains practical work on relational database systems, including database modelling, SQL querying, data exploration and Python-based database interaction.

The project covers fundamental concepts of relational databases and demonstrates the use of SQL and Python for working with structured data.

## Technologies

- SQL
- SQLite
- PostgreSQL
- Python
- psycopg2
- Flask
- ER modelling

## Key Topics

- Relational database design
- Entity-Relationship modelling
- Relational schemas
- Complex SQL queries
- Database exploration
- Python-based database interaction

## Project Structure

### Exercise 1–2: SQL Fundamentals

Introduction to relational databases, SQL queries and database operations.

### Exercise 3: Database Exploration

Implemented:
- Exploration of an existing database using SQLite
- Schema analysis
- Relational modelling
- Python scripts for database interaction

### Exercise 5: Database Design

Implemented:
- Database schema design
- SQL table definitions
- Data management tasks

## Skills Demonstrated

- Writing and optimizing SQL queries
- Understanding relational database structures
- Translating requirements into database models
- Working with databases through Python
